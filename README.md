\# 🎓 CampusGuard AI Pro



AI-Powered Multi-Agent Intelligence Platform for Students



\## 📌 Overview



CampusGuard AI Pro is an AI-powered platform designed to help students manage academic notices and verify information efficiently.



The platform consists of two intelligent agents:



\### 🎓 Academic Agent



Analyzes academic notices and helps students stay organized.



\### 📰 NewsGuard AI



Detects potentially misleading news and provides credibility analysis.

\## ⚙ Installation



\### Clone Repository



```bash

git clone <repository-url>

cd fake-news-detector

```



\### Install Dependencies



```bash

pip install -r requirements.txt

```



\### Configure Environment Variables



Create a `.env` file:



```env

GROQ\_API\_KEY=your\_api\_key\_here

```



\### Run Application



```bash

python app.py

```



Open:



http://127.0.0.1:5000
